# zombipede

Snakelike game using [Phaser](http://phaser.io/).

## Running

### Node
open a command window / terminal and run:

> node index.js

Open your browser and input the following URL:

> http://localhost:5000/

### Python

open a command window / terminal and run:

> cd ./Public

> python -m SimpleHTTPServer

Open your browser and input the following URL:

> http://localhost:8000/

## Controls

arrow keys for movement
